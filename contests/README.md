# Contests

## English

Contest solutions live here and stay separate from the reusable library. They are allowed to be problem-specific, short, and imperfect.


## Русский

Решения контестов лежат отдельно от библиотеки. Их не нужно превращать в универсальные алгоритмы: главное - не ломать рабочую логику и держать понятную структуру.
